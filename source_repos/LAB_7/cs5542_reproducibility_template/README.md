Reproducible Machine Learning Pipeline – CS 5542

Project Overview
This repository demonstrates a reproducible machine learning pipeline for CS 5542.
The goal is to ensure experiments can be reproduced reliably using deterministic
execution, controlled environments, and structured artifact generation.

Dataset
dataset/mydata/mydata.csv

Environment Setup
pip install -r requirements.txt

Running the Project
bash reproduce.sh

Outputs
logs/ – execution logs
artifacts/ – predictions and model outputs
