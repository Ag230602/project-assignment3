# Submission Screenshots

Add your application screenshots to this folder for the individual submission.

## Recommended Screenshots

1. `01_home_screen.png`
   Capture the Streamlit app loaded with the title and workflow visible.

2. `02_generated_response.png`
   Capture a completed run showing the explanation, retrieved context, and warehouse facts.

3. `03_monitoring_or_evaluation.png`
   Capture either the monitoring section with health or metrics, or the sample evaluation output.

## Suggested Steps

1. Start the backend:
   `uvicorn api:app --reload`
2. Start the frontend:
   `streamlit run app.py`
3. Open the app in your browser.
4. Take the three screenshots above and save them in this folder.

## Optional File Names

- `01_home_screen.png`
- `02_generated_response.png`
- `03_monitoring_or_evaluation.png`

## Submission Note

If you do not have a public application link, these screenshots can be used as application evidence in the submission.
